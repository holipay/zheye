## 2. 缓存集成

### 2.1 修复 cache.py

**文件**: `app/cache.py`

修复 `set_cached` 函数的 ttl 参数无效问题：

```python
def set_cached(key: str, value: Any, ttl: int = None):
    if ttl:
        cache = TTLCache(maxsize=1, ttl=ttl)
        cache[key] = value
        _cache[key] = cache[key]
    else:
        _cache[key] = value
```

## 7. AI 分析调整

### 7.1 从每日分析改为周报/月报

**原因**:
- 每日新闻量不足（约24条/天）
- 减少 AI API 调用成本
- 周/月数据更充足，洞察更准确

### 7.2 新增周报/月报脚本

**文件**: `scripts/run_period_report.py`

使用方法：
```bash
# 生成周报
python scripts/run_period_report.py --period weekly --date 2026-06-01

# 生成月报
python scripts/run_period_report.py --period monthly --date 2026-06-01
```

### 7.3 新增 API 端点
